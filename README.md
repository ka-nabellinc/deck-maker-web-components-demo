# deck-maker-web-components-demo

2024年2月10日 採用イベント用レポジトリです。


```
/src
└ /static <- 画像ファイルの格納場所
└ /components <- Web Componentsの格納場所
└ entry.ts <- 全てのWeb Componentsを読み込むための単一エントリーポイント
```

## 環境構築

```sh
npm i
npm run storybook
```

## Unit Testing

```sh
npm run test:unit
```

## ビルド

```sh
npm run build
```
